Most teams run a match off a laptop balanced on a cart. This is a purpose-built driver
station: a fixed control layout, integrated [HawkLights](/projects/hawklights), and a
housing built to take a beating in the pit.

## Explore the build

Hover any label to isolate that part of the station.



## The build

The whole thing is a single machined backplate that everything else mounts to — no
loose parts rattling around in a bag between matches.

![Machined backplate](/projects/driver-station/backplate.png)

Status LEDs run along the top edge so the drive team can read robot state from across
the pit without looking at a screen.

![Integrated status LEDs](/projects/driver-station/leds.png)

The main display is fixed at a readable angle, and the network drops are broken out to
a labelled panel so swapping a bad cable mid-event is a ten-second job.

![Fixed display and network panel](/projects/driver-station/display.png)

![Ethernet breakout](/projects/driver-station/ethernet.png)
